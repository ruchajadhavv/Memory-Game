How the Game Works
The game contains emoji pairs (🍎 🍌 🍇 🍓 🍊 🥑 🍑 🍒)
Cards are shuffled randomly at the start
Player flips two cards at a time:
✅ If they match → they stay open
❌ If not → they flip back
Game ends when all pairs are matched
A win message is displayed with total moves
